Oppose Inertia

Fri Dec 11, 2015 in using tags

Oppose Inertia was an experiment to make a 2D space combat game with a heavy focus on procedural generation. It never turned out exactly as I’d hoped but it was a good learning experience. Explore star systems, each with unique enemies, which drop new weapons, abilities, or ship schematics for you to use.
